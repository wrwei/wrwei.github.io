Module 1: What is an AI Agent?

Understanding what AI agents are, how they differ from chatbots, and the core components that make them work.

1

Definition

An AI agent is a software system that uses a large language model (LLM) as its reasoning core, combined with tools, memory, and planning capabilities, to autonomously complete multi-step tasks on behalf of a user.

2

Agent vs Chatbot

A chatbot responds to a single prompt. An agent reasons across multiple steps, decides which tools to use, maintains context across interactions, and takes actions in the real world — calling APIs, reading files, querying databases.

AspectChatbotAgent
ReasoningSingle turnMulti-step
ToolsNoneSearch, APIs, code
MemorySession onlyShort + Long term
ActionsText outputReal-world effects
AutonomyReactiveGoal-directed
3

Core Components

🧠 LLM Core

The reasoning engine — GPT-4o, Claude Sonnet 4, etc.

🔧 Tools

Functions the agent can invoke: search, calculate, API calls.

💾 Memory

Short-term (conversation) and long-term (persistent storage).

📋 Planning

Decomposing complex goals into executable sub-tasks.

🔄 Orchestration

The control loop that ties everything together.

4

The Agent Landscape (2025-2026)

The AI agent ecosystem has exploded. Key protocols like MCP (Model Context Protocol) and A2A (Agent-to-Agent) provide standardised communication. Major cloud providers (AWS, Google Cloud, Azure) now offer agent marketplaces. The market is projected to reach $50B by 2030.

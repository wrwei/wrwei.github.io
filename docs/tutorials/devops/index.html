DevOps

Ten modules taking you from your first terminal command to a working CI/CD pipeline with containers. Designed for first-year undergraduates at Lancaster University.

10 modules Level 4 · FHEQ ~8 hrs total
Module 01
Introduction to DevOps and the Unix Environment
Understand what DevOps is, why it exists, and take your first steps in the terminal.
Foundations
Module 02
Unix Tools and Simple Automation
Learn to search, combine and process text with Unix tools, then write your first shell script.
Unix & CLI
Module 03
Version Control and Git Basics
Understand why version control exists and master the core Git commands you will use every day.
Version Control
Module 04
Collaborating with Git
Learn to work with branches, remote repositories and pull requests — the everyday workflow of professional development teams.
Version Control
Module 05
Continuous Integration and Automated Testing
Understand what CI is, what automated tests look like, and how to read pipeline output.
CI/CD
Module 06
CI Pipelines in Practice with GitHub Actions
Understand pipeline structure and learn to read, modify and debug a real GitHub Actions workflow.
CI/CD
Module 07
Automating Repetitive Tasks
Stop typing the same commands every day. Write automation scripts that work identically locally and in CI.
Automation
Module 08
Virtualisation and Containers
Understand the 'works on my machine' problem and learn to package applications with Docker.
Containers
Module 09
Containers in a DevOps Workflow
Connect Docker to your CI pipeline and complete the journey from git push to a verified, containerised application.
Containers
Module 10
DevOps in Practice and Module Review
Trace the full DevOps pipeline end to end, explore the culture behind the tools, and prepare for your exam.
Foundations

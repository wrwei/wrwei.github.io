MCP Protocol Integration

MCP协议集成

8.1 What is MCP?

什么是MCP？

The Model Context Protocol (MCP) is an open standard created by Anthropic for connecting AI agents to external tools and data sources. Think of it as USB-C for AI — one standard connector that works with everything. It was donated to the Linux Foundation in December 2025 and now has 97M+ monthly SDK downloads.

模型上下文协议（MCP）是Anthropic创建的开放标准，用于将AI智能体连接到外部工具和数据源。可以把它想象成AI的USB-C——一个通用标准连接器。它于2025年12月捐赠给Linux基金会，目前每月SDK下载量超过9700万次。

架构

8.3 Building a Simple MCP Server

构建简单MCP服务器
pip install mcp
from mcp.server.fastmcp import FastMCP

mcp = FastMCP('My Agent Tools')

@mcp.tool()
def get_stock_price(symbol: str) -> str:
    """Get the current stock price for a ticker symbol."""
    prices = {'AAPL': 227.50, 'GOOGL': 178.30, 'MSFT': 445.20}
    price = prices.get(symbol.upper(), 'Unknown')
    return f'{symbol}: ${price}'

@mcp.tool()
def calculate(expression: str) -> str:
    """Safely evaluate a mathematical expression."""
    import simpleeval
    try:
        result = simpleeval.simple_eval(expression)
        return str(result)
    except Exception as e:
        return f'Error: {e}'

mcp.run(transport='stdio')
⚠️Never use eval() in production. Use a safe math parser like simpleeval or asteval. / 生产环境中切勿使用eval()。请使用simpleeval或asteval等安全数学解析器。

8.4 Connecting to Claude Desktop

连接到Claude Desktop

Add your server to Claude Desktop's config file (claude_desktop_config.json):

将您的服务器添加到Claude Desktop的配置文件：

{
  "mcpServers": {
    "my-tools": {
      "command": "python",
      "args": ["path/to/your/server.py"]
    }
  }
}

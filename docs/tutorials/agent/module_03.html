3.1 OpenAI — Your First Call (GPT-4o)

OpenAI — 第一次调用

The simplest possible call to OpenAI's Chat Completions API:

最简单的OpenAI聊天完成API调用：

from openai import OpenAI

client = OpenAI()  # reads OPENAI_API_KEY from env

response = client.chat.completions.create(
    model='gpt-4o',
    messages=[
        {'role': 'system', 'content': 'You are a helpful assistant.'},
        {'role': 'user', 'content': 'What is an AI agent?'}
    ]
)

print(response.choices[0].message.content)

3.2 Anthropic — Your First Call (Claude Sonnet 4)

Anthropic — 第一次调用

The equivalent call using Anthropic's Messages API:

使用Anthropic Messages API的等效调用：

import anthropic

client = anthropic.Anthropic()  # reads ANTHROPIC_API_KEY from env

message = client.messages.create(
    model='claude-sonnet-4-20250514',
    max_tokens=1024,
    system='You are a helpful assistant.',
    messages=[
        {'role': 'user', 'content': 'What is an AI agent?'}
    ]
)

print(message.content[0].text)
